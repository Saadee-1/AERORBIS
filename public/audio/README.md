# AeroVerse Audio Files

Place your ambient audio files in this directory with the following names:

## Required Audio Files

1. **home-ambient.mp3** - Launch pad ambience: deep engine hum, soft wind, distant radio static
2. **learn-ambient.mp3** - Orbital serenity: calm harmonic drones, gentle chime pulses
3. **research-ambient.mp3** - Analytical mood: low rhythmic data pings, sonar blips
4. **tools-ambient.mp3** - Workshop vibe: soft interface beeps, holographic whirr
5. **community-ambient.mp3** - Connection ambience: radio bursts, Morse-like tones
6. **dashboard-ambient.mp3** - Mission control: radar sweeps, subtle air circulation hum

## Audio Specifications

- **Format**: MP3 (recommended) or WAV
- **Duration**: 30-120 seconds (will loop seamlessly)
- **Sample Rate**: 44.1 kHz or 48 kHz
- **Bit Rate**: 192 kbps or higher
- **Volume**: Normalize to -12 dB to -6 dB

## Recommended Sources for Royalty-Free Space Ambience

- [Freesound.org](https://freesound.org) - Search for "space ambient", "drone", "atmosphere"
- [Incompetech](https://incompetech.com) - Ambient music by Kevin MacLeod
- [Purple Planet](https://www.purple-planet.com) - Space/ambient tracks
